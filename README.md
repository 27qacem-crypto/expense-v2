# تطبيق تسجيل المصروفات

تطبيق PWA لتسجيل مصروفات الموظفين مع رفع صور الإيصالات.

## الإعداد

1. افتح `Code.gs` في Google Apps Script واربطه بـ Google Sheet
2. انشر الـ Script كـ Web App (Execute as: Me, Access: Anyone)
3. انسخ رابط الـ Web App
4. افتح `app.js` وحط الرابط مكان `___SCRIPT_URL___`
5. نزل أيقونات `icon-192.png` و `icon-512.png` من GitHub أو استخدم أي أيقونات
6. ارفع الملفات على أي hosting (GitHub Pages, Netlify, إلخ)
